# Áudio Ambiente do Codex

## Arquivo Necessário

### **ambiente.mp3**
Música/som ambiente para criar atmosfera gótica e misteriosa

## Características Recomendadas:
- **Duração**: 2-5 minutos (loop)
- **Estilo**: Ambient, dark, gótico, mistérico
- **Volume**: Baixo/médio (será reproduzido a 20% no código)
- **Tom**: Sombrio, contemplativo, atmosférico

## Sugestões de Estilo Musical:
- Ambient dark
- Drone music
- Gothic atmosphere
- Dark classical
- Música de meditação sombria
- Sons de vento, sussurros distantes
- Melodias de órgão ou violoncelo

## Onde Encontrar Áudio Gratuito:

### Sites de Música Livre:
1. **FreePD** - https://freepd.com (domínio público)
2. **Free Music Archive** - https://freemusicarchive.org
3. **Incompetech** - https://incompetech.com (Kevin MacLeod)
4. **YouTube Audio Library** - música sem copyright
5. **Pixabay Music** - https://pixabay.com/music/
6. **Bensound** - https://www.bensound.com

### Termos de Busca:
- "dark ambient"
- "gothic atmosphere"
- "horror ambience"
- "mysterious background music"
- "dark drone"
- "cinematic dark"

## Alternativa
Se não tiver áudio, o botão de controle de áudio continuará visível mas não tocará nada. O site funcionará normalmente mesmo sem o arquivo de áudio.

## Conversão para MP3
Se tiver áudio em outro formato:
- Use **Audacity** (gratuito) para converter
- Ou conversor online: https://online-audio-converter.com/
